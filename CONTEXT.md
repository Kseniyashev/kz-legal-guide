# Гид: открытие ТОО + Astana Hub в Казахстане

**Репо:** github.com/Kseniyashev/kz-legal-guide  
**Сайт:** https://kseniyashev.github.io/kz-legal-guide/  
**Стек:** один HTML-файл, vanilla JS, без фреймворков, данные в localStorage

---

## Что уже сделано

Пошаговый веб-мастер из 9 шагов:

| Шаг | ID | Содержание |
|-----|----|------------|
| 1 | `intro` | Введение, обзор процесса |
| 2 | `visa` | Статус бизнес-иммигранта — въезд, регистрация, ИИН, **РВП с основанием «бизнес-иммигрант»**, таймлайн |
| 3 | `data` | Форма: название ТОО, адрес, ОКЭД, капитал, директор, учредители, описание бизнеса |
| 4 | `docs` | Генерация Устава и Решения учредителя / Протокола — с данными из шага 3 |
| 5 | `register` | Регистрация ТОО: онлайн (egov.kz) или ЦОН, чеклист, таймлайн |
| 6 | `postreg` | После регистрации: банк, налоги, статкомитет, сравнение налоговых режимов |
| 7 | `hubinfo` | Что такое Astana Hub, льготы (0% КПН, 0% НДС), кто может подать, процесс |
| 8 | `hubapp` | Заявка в Hub — предзаполнена из данных шага 3, кнопка «Скопировать черновик» |
| 9 | `final` | Финальный чеклист всего процесса с прогресс-баром |

## Контекст и уточнения

**Учредители — граждане России.** Всё заточено под ЕАЭС:
- Виза не нужна, разрешение на работу не нужно
- Нужны: регистрация по месту пребывания → ИИН → ТОО → РВП
- Справка о несудимости из МВД РФ: нужен **только апостиль**, перевод не нужен (документ на русском, русский — официальный язык в Казахстане)
- РВП выдаётся на 1 год, продлевается ежегодно пока действует ТОО

**Что уточнили в процессе:**
- РВП с основанием «бизнес-иммигрант» — отдельный обязательный шаг после регистрации ТОО (не ВНЖ)
- Нотариальный перевод справки о несудимости не нужен — убрали
- Перевод на казахский тоже не нужен

## Архитектура файла

`index.html` — всё в одном файле:
- CSS в `<style>` (~150 строк)
- HTML-каркас: header + sidebar + main
- JS в `<script>`:
  - `STEPS[]` — массив шагов
  - `DEFAULT_DATA` / `data` — состояние, хранится в localStorage
  - `render()` — точка входа, вызывает нужный `stepXxx()`
  - Каждый шаг — отдельная функция `stepIntro()`, `stepVisa()`, `stepData()` и т.д.
  - `genUstav()` / `genReshenie()` — генерация документов из шаблонов
  - `toggleCheck(id)` — интерактивные чеклисты с сохранением в localStorage
